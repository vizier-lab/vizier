# Calculator Skill

This skill enables you to perform mathematical calculations and computations.

## Capabilities

- Basic arithmetic operations (addition, subtraction, multiplication, division)
- Complex mathematical expressions
- Unit conversions
- Statistical calculations

## Usage

When the user asks for mathematical calculations, use this skill to:

1. Parse the mathematical expression
2. Perform the calculation
3. Return the result with appropriate precision

## Examples

- "What is 15% of 200?"
- "Calculate the square root of 144"
- "Convert 100 Fahrenheit to Celsius"
- "What is the average of these numbers: 10, 20, 30, 40, 50?"

## Guidelines

- Always show your work when performing complex calculations
- Use appropriate precision for the result
- For unit conversions, specify the conversion formula used
- If the calculation is ambiguous, ask for clarification
